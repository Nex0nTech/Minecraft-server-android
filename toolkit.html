What Each Menu Option Does

  • 1) Install SDK Platform Tools: Installs official Android Platform Tools for ADB/Fastboot.
  • 2) Check ADB connection: Verifies connected device via ADB.
  • 3) Reboot to bootloader: Reboots device into fastboot mode.
  • 4) Reboot to recovery: Reboots device into recovery mode.
  • 5) Install APK from PC: Installs APK from PC to device.
  • 6) Uninstall package (debloat): Removes unwanted packages safely.
  • 7) List installed packages: Lists all installed packages.
  • 8) Pull file from device: Copies a file from device to PC.
  • 9) Push file to device: Copies a file from PC to device.
  • 10) USB debugging instructions: Shows how to enable USB debugging.
  • 11) Check Fastboot connection: Verifies fastboot connection.
  • 12) Flash boot.img: Flashes the boot partition.
  • 13) Flash recovery.img: Flashes recovery partition.
  • 14) Extract boot.img from payload.bin: Extracts images from payload.bin.
  • 15) Show device info: Displays device info.
  • 16) Clear ADB keys: Resets ADB keys and restarts server.
  • 17) Restart ADB server: Restarts ADB server.
  • 18) Exit: Closes the toolkit.
