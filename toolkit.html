The NexonTech Android Toolkit is a Windows batch utility designed to simplify ADB and Fastboot operations. Instead of typing commands manually, the user interacts with a clean, color-coded menu. Below is the exact structure of the toolkit menu and a full explanation of every option.

What Each Menu Option Does

1) Install SDK Platform Tools

Downloads and installs official Android Platform Tools, enabling ADB and Fastboot commands.

2) Check ADB connection

Verifies if the device is detected via ADB.

3) Reboot to bootloader

Reboots the device into fastboot/bootloader mode.

4) Reboot to recovery

Reboots the device into recovery mode.

5) Install APK from PC

Installs an APK from the computer onto the device.

6) Uninstall package (debloat)

Removes a package for the current user. Use carefully.

7) List installed packages

Shows all installed package names.

8) Pull file from device

Copies a file from the device to the PC.

9) Push file to device

Copies a file from the PC to the device.

10) USB debugging instructions

Displays steps to enable USB debugging.

11) Check Fastboot connection

Verifies fastboot device detection.

12) Flash boot.img

Flashes the boot partition. Using the wrong file can cause bootloops.

13) Flash recovery.img

Flashes the recovery partition. Wrong files may brick the device.

14) Extract boot.img from payload.bin

Extracts images from payload.bin files used in modern ROMs.

15) Show device info

Displays detailed device and system information.

16) Clear ADB keys

Resets ADB authorization keys and restarts the ADB server.

17) Restart ADB server

Restarts the ADB server without removing keys.

18) Exit

Closes the toolkit.
