Running a Minecraft Server on Termux (Android)

A simple, copy paste friendly guide for hosting Minecraft Java 1.19 on your phone.

Overview

This guide walks you through setting up a Minecraft Java Edition 1.19 server using Termux on Android. Every command is ready to copy with a single click, and each step is explained so you understand what you are doing, not just pasting blindly.

Note: You should already have Termux installed from a trusted source. Make sure your device has enough free storage and a stable internet connection.

Step 1, Update and upgrade Termux packages

Before installing anything, it is good practice to update the package lists and upgrade existing packages. This ensures you get the latest versions and reduces compatibility issues. 

pkg update && pkg upgrade -y

What this does:

  • pkg update refreshes the list of available packages.
  • pkg upgrade -y upgrades installed packages to their latest versions without asking for confirmation.

Step 2, Install required packages

The Minecraft server requires Java to run. We also install wget to download the server file and nano as a simple text editor in case you want to edit configuration files later. 

pkg install openjdk-17 wget nano -y

Package breakdown:

  • openjdk-17 Java 17, required to run modern Minecraft servers.
  • wget Command line tool to download files from the internet.
  • nano Lightweight text editor for editing configuration files directly in Termux.

Step 3, Create the server folder

To keep things organized, you create a dedicated directory for your Minecraft server. This is where the server JAR, world data, and configuration files will live. 

mkdir ~/mcserver
cd ~/mcserver

Explanation:

  • mkdir ~/mcserver creates a folder named mcserver in your home directory.
  • cd ~/mcserver moves you into that folder so all future commands run inside it.

Step 4, Download the Minecraft 1.19 server JAR

Now you download the official Minecraft server JAR for version 1.19 using wget. The file will be saved as minecraft_server.1.19.jar inside your mcserver folder. 

wget https://piston-data.mojang.com/v1/objects/e00c4052dac1d59a1188b2aa9d5a87113aaf1122/server.jar -O minecraft_server.1.19.jar

What is happening here:

  • The long URL points to the official Mojang server JAR for Minecraft 1.19.
  • -O minecraft_server.1.19.jar tells wget to save the file with a clear versioned name.
Tip: If Mojang updates the server or the link changes in the future, you may need a new URL.

Step 5, Accept the EULA

Minecraft servers require you to accept the End User License Agreement EULA before they can run. Normally, the server generates a eula.txt file on first launch and you edit it manually. Here, you create it directly and set it to true. 

echo "eula=true" > eula.txt

Explanation:

  • echo "eula=true" prints the text eula=true.
  • > eula.txt writes that text into a file named eula.txt, creating or overwriting it.
Important: By setting eula=true, you confirm that you have read and accepted Mojang EULA. You should review it on the official Minecraft website.

Step 6, Start the Minecraft server

With Java installed, the server JAR downloaded, and the EULA accepted, you are ready to start the server. The command below launches the server with specific memory settings and garbage collection options. 

java -Xms512M -Xmx1789M -XX:+UseG1GC -XX:MaxGCPauseMillis=100 -jar minecraft_server.1.19.jar nogui

Flag breakdown:

  • -Xms512M sets the minimum RAM for the server to 512 MB.
  • -Xmx1789M sets the maximum RAM to 1789 MB about 1.7 GB. Adjust if your device has more or less memory.
  • -XX:+UseG1GC uses the G1 garbage collector, which is generally good for Minecraft servers.
  • -XX:MaxGCPauseMillis=100 aims to keep garbage collection pauses under 100 ms.
  • -jar minecraft_server.1.19.jar tells Java which JAR file to run.
  • nogui runs the server without the graphical interface perfect for Termux.
First launch: On the first run, the server generates world files and configuration files. This can take a bit of time, be patient. Once it says Done in the console, your server is up.

Connecting to your server

Once the server is running, you can connect from Minecraft Java Edition on another device using your phone local IP address and the default port 25565, as long as your network and firewall settings allow it.

If you want friends outside your network to join, you need to look into port forwarding, VPNs, or tunneling services, those are more advanced topics and depend on your router and setup.
